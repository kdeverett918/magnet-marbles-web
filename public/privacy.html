Magnet Marbles

Privacy

Last updated: June 22, 2026

Summary

Magnet Marbles is a browser game prototype. The game does not use ads, analytics trackers, user accounts, payment processing, in-app purchases, loot boxes, or cross-site tracking.

Data Stored On Your Device

The game uses browser local storage for gameplay convenience:

  • Sound on/off, SFX volume, haptics, bot difficulty, and quality settings.
  • First-run tutorial completion.
  • Offline progression, stars, daily challenge completion, and selected trail cosmetic.

This data stays in your browser unless you use Reset data in the game's Options menu, clear site data, or your browser syncs it under settings you control.

Online Play

Online matches connect to the Magnet Marbles game server through WebSockets. The server receives temporary session identifiers, room information, selected mode, and live gameplay input needed to run the match. The current game does not create accounts or grant online progression rewards.

Fair Play And Purchases

The current web build has no purchases, ads, subscriptions, rewarded videos, or paid cosmetics. Offline stars and cosmetics are local-only progression for playtesting; online matches do not award account progression.

Third Parties

Hosting providers may process standard web server logs such as IP address, request time, user agent, and error information for security and reliability. Magnet Marbles does not add third-party analytics, ad SDKs, or social trackers.

Children

The game is designed as an arcade game and does not knowingly collect personal information from children. Do not enter personal information into room codes or support reports.

Control

You can reset local game data from Options in the main menu with the two-step Reset data control, or by clearing site data for this domain in your browser settings. If you report a bug, avoid sharing personal details and do not include private information in room codes.
