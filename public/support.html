Support

Last updated: June 22, 2026

Fast Fixes

  • If the game feels slow on a phone, open Options in the menu and choose Lite.
  • If the game is too loud, lower the SFX volume slider from the menu or pause options, then use Test SFX.
  • If audio is silent, tap the screen once, then toggle Sound off and on.
  • If haptics feel distracting, use the Haptics toggle in the menu or pause options; use Test Haptics to preview the vibration on supported phones.
  • If you want a clean profile, use Reset data in Options and confirm the reset.
  • If online play stalls, wait up to 60 seconds for the server to wake, then retry.
  • If the app looks stale after a deploy, refresh the page or clear site data.
  • If a reviewer asks for the Candidate ID, use the small Candidate stamp in the menu and compare it with build.json.
  • If the game shows Table reset needed, include the crash support code that starts with MM-.

Controls

  • Touch: drag anywhere on the playfield to roll.
  • Touch: hold the lower-right thumb zone or Magnet button to pull marbles.
  • Touch: tap or flick the lower-right thumb zone, or tap Dash, to pulse forward.
  • Keyboard: WASD or arrows move, Space holds magnet, Shift dashes, E uses a powerup.

What To Include In A Bug Report

  • The menu Candidate ID: commit and source fingerprint.
  • Any crash support code shown on the Table reset needed screen.
  • Device model, browser, and operating system.
  • Game mode, online room code if relevant, and what happened before the issue.
  • A screenshot or screen recording when possible.
  • Whether High or Lite quality was selected.
  • Whether Sound, SFX volume, Test SFX, Haptics, or Test Haptics changed the issue.

Do not include personal information in room codes, screenshots, or bug reports. The current web build has no purchases, ads, subscriptions, or paid cosmetics.

Crash Recovery

If the game recovers to a Table reset needed screen, try Reload first. If it repeats, include the visible crash support code, menu Candidate ID, and device details in the report. The Reset local data button only clears Magnet Marbles settings, tutorial progress, stars, daily records, and cosmetics for this site.

Data Reset

To reset settings, tutorial state, stars, daily challenge completion, daily streaks, and skin/trail unlocks, use the two-step Reset data control in Options. You can also clear site data for this domain in your browser settings.

Candidate ID

The main menu shows a compact Candidate stamp with the current commit and source fingerprint. For release QA, capture that stamp in screenshots or videos and verify it matches this build's build.json. This keeps phone, online, and human review evidence tied to the exact candidate that was tested.
